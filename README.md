# Grenade_Weapons

Give your weapon an ability to shoot grenades


# Commands:

sm_grenadeweapon (

sm_gw



# Cvars:
 
sm_gw_version (Plugin Version)

sm_enable_gweapon (Enable/disable the plugin)

sm_grenade_w (Grenade type to spawn Accepted grenade types: flash, heg, molotov)
